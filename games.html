games

I started making games in 2005. They started off as fun storytelling devices for me, and became so much more. I enjoy using Twine, Ren'Py and PICO-8. My current project is Terranova. I do game development streaming on Twitch sometimes. Check out my Twitch for more info.

Terranova

Terranova is a nostalgic interactive fiction game set in the early 2000's following four fangirls and their roleplay community on LimeJournal.
